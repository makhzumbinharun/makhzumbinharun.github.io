# Makhzum Bin Harun — Portfolio Website

> Personal portfolio of a CSE student, C/C++ programmer, and competitive programmer based in Dhaka, Bangladesh.

🌐 **Live site:** [makhzumbinharun.github.io](https://makhzumbinharun.github.io)

---

## ✨ Features

- **Dark / Light mode** — persists across sessions via localStorage, respects system preference on first visit
- **Typing animation** — cycles through roles in the hero section
- **Scroll reveal** — elements animate into view as you scroll
- **Competitive Programming section** — profiles across 7 platforms (Codeforces, LeetCode, CodeChef, HackerRank, VJudge, BeeCrowd, ICPC)
- **GitHub activity stats** — dynamic stats and streak cards that adapt to the current theme
- **Project filter** — filter projects by category (Technical, Social Impact, Business)
- **Experience timeline** — full work history with logos and bullet points
- **Contact form** — opens default mail client with pre-filled message
- **Fully responsive** — works on mobile, tablet, and desktop
- **No frameworks** — pure vanilla HTML, CSS, and JavaScript

## 🚀 Running Locally

No build step needed — just clone and open.

```bash
git clone https://github.com/makhzumbinharun/makhzumbinharun.github.io.git
cd makhzumbinharun.github.io
# Open index.html in your browser
```

Or use VS Code's **Live Server** extension for hot reload during development.
